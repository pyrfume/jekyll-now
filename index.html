The Pyrfume project aims to make odorant-linked data more available, more usable, more visible, and more standardized across researchers in the experimental and theoretical olfaction research community. It consists of:

  • pyrfume, a python package [source] [tutorials] [api docs]
  • pyrfume-data, a collection of curated and standardized datasets for research and model benchmarking [public source] [curation status]
  • rfume, an R package with limited features [source]
  • pyRESTfume, a REST API for state-less computations with odorant-linked data
