# The Pyrfume Project


## Mission:
Pyrfume aims to make odorant-linked datasets:
- Easy to [F]ind
- Easy to [A]ccess and use
- [I]nteroperable with other datasets and tools
- Facilitate [R]eproducible analysis and modeling
